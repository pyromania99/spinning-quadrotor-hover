# Spinning Quadrotor Hover

This repository contains code and experimental resources for the paper:

"Spinning Hover for Lift Augmentation in Quadrotor Platforms"

## Contents

* PX4 modifications (yaw reference handling for spinning flight)
* Simulation scripts (MuJoCo + BEMT)
* Hardware configuration notes

## Notes

This is an initial release of research code. A cleaned and fully documented version will be released in future updates.
